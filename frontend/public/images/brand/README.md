# Brand Images Directory

This directory contains the three brand images displayed on the homepage with 3D rotation effect.

## Required Images

1. **grandson-ent-logo.png** - Grandson Entertainment Logo (first image)
2. **made-in-guinea-label.png** - Made in Guinea Label (second image)  
3. **horse-mascot.png** - Grandson Horse Mascot (third image)

## Image Requirements

- **Format**: PNG, JPG, or WebP
- **Size**: Maximum 2MB per image
- **Dimensions**: Square aspect ratio recommended (e.g., 500x500px, 800x800px)
- **Background**: Transparent PNG recommended for best results

## How to Add Your Images

1. Save your three images with the exact filenames listed above
2. Place them in this directory (`frontend/public/images/brand/`)
3. The images will automatically appear on the homepage with 3D rotation effect
